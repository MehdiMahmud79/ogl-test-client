# OGL Test Client

A scalable, maintainable Angular application built with TypeScript, following modern best practices for performance and accessibility.

## Features

- Standalone Angular components
- Signals-based state management
- Lazy-loaded feature routes
- Strict TypeScript configuration
- Reactive forms
- Optimized static images with `NgOptimizedImage`

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [Angular CLI](https://angular.io/cli)

> **Note:**  
> <span style="color: yellow;">Update the `googleMapsApiKey` inside the environment file --if it is expired--.</span>

### Installation

```bash
npm install -g @angular/cli@latest
npm install
```

### Development Server

```bash
ng serve
or
npm run start
```

Navigate to `http://localhost:4200/`. The app will reload if you change any source files.

### Build

```bash
ng build
or
npm run build
```
